## 🆘 Problemas e Soluções

### P: O site está lento!

**R:** Experimente:

1. **Limpar cache:**
   - Chrome: Ctrl+Shift+Delete
   - Firefox: Ctrl+Shift+Delete

2. **Desabilitar extensões** — Pode estar bloqueando

3. **Testar internet:**
   - Speedtest.net
   - Idealmente: 5+ Mbps

4. **Reportar:** Se muito lento, avise suporte

---

### P: Produto não aparece nos resultados!

**R:** Verifique:

1. **Filtros ativos?** → Clique "Limpar filtros"
2. **Categoria correta?** → Mude para "Todos"
3. **Preço muito baixo?** → Aumento limite de preço
4. **Digite certo?** → Sem acentos obrigatórios

Se ainda não aparecer → Contate suporte.
