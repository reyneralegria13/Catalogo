## 🎯 Modelo de Vendas

### Tipo: Modelo Híbrido B2B/B2B2C

```
1) Template Próprio (B2B)
[TupãSoft] → Venda direta → [Cliente]

2) Revenda de Parceiros (B2B2C)
[Fornecedor] ← Comissão → [TupãSoft] → Venda → [Cliente]

3) Projeto Personalizado (B2B)
[Cliente] → Especificação → [TupãSoft] → Entrega sob medida
```

### Regras Comerciais por Trilha

- **Template próprio:** precificação definida pela TupãSoft, com receita direta.
- **Revenda:** preço e contrato definidos com parceiro; TupãSoft recebe comissão por venda/renovação.
- **Personalizado:** proposta técnica/comercial por escopo, prazo e nível de integração.

### Fluxo da Compra — Perspectiva do Cliente

```
1. DESCOBERTA (Dia 1)
   ├─ Usuário busca: "software supermercado"
   ├─ Clica em produto
   └─ Vê detalhes, reviews, preço

2. AVALIAÇÃO (Dia 1-3)
   ├─ Compara com outros produtos
   ├─ Verifica reviews e rating
   ├─ Solicita trial via WhatsApp
   └─ Testa por 7-14 dias

3. DECISÃO (Dia 3-7)
   ├─ Clica "Falar no WhatsApp"
   ├─ Envia carrinho de compras
   └─ Equipe responde com opções pagamento

4. COMPRA (Dia 7-10)
   ├─ Escolhe forma pagamento (cartão, PIX, boleto)
   ├─ Realiza pagamento
   └─ Recebe acesso imediatamente

5. ONBOARDING (Dia 10-15)
   ├─ Acesso aos arquivos/sistema
   ├─ Suporte de implementação
   ├─ Treinamento básico
   └─ Vai independente ✅

6. SUCESSO (Dia 15+)
   ├─ Usa software dia-a-dia
   ├─ Recebe suporte contínuo
   └─ Renovação automática (se mensal)
```

### Fluxo Comercial — Software Personalizado

```
1. Descoberta e briefing
     ├─ Cliente explica necessidade
     └─ TupãSoft levanta requisitos e restrições

2. Diagnóstico e proposta
     ├─ Escopo funcional
     ├─ Estimativa de prazo e custo
     └─ Modelo contratual e marcos

3. Contratação
     ├─ Assinatura de contrato
     ├─ Pagamento de entrada
     └─ Planejamento do projeto

4. Entrega e acompanhamento
     ├─ Entregas por sprint/marco
     ├─ Validação com cliente
     └─ Suporte pós-implantação
```

## 💰 Modelo de Comissão

### Estrutura Tiered

```
Comissão baseada em volume:

Faixas:
├─ 1-10 vendas/mês    → 20% comissão
├─ 11-50 vendas/mês   → 25% comissão
├─ 51+ vendas/mês     → 30% comissão
└─ Parceria exclusiva → 35% comissão
```

### Exemplo: SuperMarket Total

```
Preço mensal: R$ 799

Cenário 1: Poucos clientes
├─ 5 clientes × R$ 799 × 20% = R$ 799/mês
└─ Receita anual = R$ 9.588

Cenário 2: Muitos clientes
├─ 30 clientes × R$ 799 × 25% = R$ 5.992/mês
└─ Receita anual = R$ 71.920

Cenário 3: Parceria exclusiva
├─ 50 clientes × R$ 799 × 30% = R$ 11.985/mês
└─ Receita anual = R$ 143.820
```

### Produtos Licença vs. Recorrente

```
LICENÇA ÚNICA (pago uma vez)
├─ Preço: R$ 297 (CalcPro Fiscal)
├─ Comissão: 20% = R$ 59.40 (uma vez)
└─ Exemplo: 5 vendas/mês = R$ 297

RECORRENTE (pago todo mês)
├─ Preço: R$ 799 (SuperMarket)
├─ Comissão: 20% = R$ 159.80 (todo mês)
└─ Exemplo: 5 clientes = R$ 799/mês (recorrente)

👉 Melhor negócio: Softwares recorrentes
```

## 🔄 Fluxo Operacional

### Passo 1: Cliente Chega ao Site

```
Cliente → Google/Facebook → TupãSoft.com
          ↓
      Vê catálogo
          ↓
      Clica produto interessante
```

### Passo 2: Cliente Adiciona ao Carrinho

```
Modal de Detalhes
├─ Título, descrição, features
├─ Rating (⭐) e reviews
├─ Preço e modelo (mês/licença)
└─ Botões:
   ├─ [🛒 Adicionar ao Carrinho]
   └─ [💬 Falar no WhatsApp]
```

### Passo 3: Cliente Envia via WhatsApp

```
TupãSoft envia link do WhatsApp:
https://wa.me/5592999990000?text=...

Mensagem Pré-formatada:
"Olá, TupãSoft! Gostaria de adquirir:

• SuperMarket Total — R$ 799/mês
• CalcPro Fiscal — R$ 297/licença

Total: R$ 1.096

Aguardo contato!"
```

### Passo 4: Equipe TupãSoft Recebe

```
WhatsApp Desktop/Web
     ↓
Notificação: Nova mensagem
     ↓
Atendente verifica:
├─ Quantos produtos?
├─ Qual orçamento?
├─ Quando precisa?
├─ Qual tamanho empresa?
└─ Precisa de trial?
```

### Passo 5: Qualificação do Lead

**Perguntas de Qualificação:**

```
Atendente: Olá! Obrigado pelo interesse! 🎉

Alguns esclarecimentos rápidos:

1️⃣ Quantos funcionários tem sua empresa?
2️⃣ Vai usar este software para qual função?
3️⃣ Quando precisa começar?
4️⃣ Você quer testar antes de comprar?

Cliente: [Respostas] ✉️
```

**Classificação:**

- 🔥 **Quente:** Pronto para comprar agora
- 🟡 **Morno:** Interessado, mas quer testar
- ❄️ **Frio:** Pesquisando, sem urgência

### Passo 6: Oferta Customizada

**Se Quente:**

```

SuperMarket Total:
✅ Melhor para empresas de seu porte
✅ Suporte local em Manaus
✅ De implementação 48h

1. Cartão crédito (débito imediato)
2. PIX (débito imediato)
3. Boleto (3-5 dias)
4. Parcelas (3x sem juros)

Qual te interessa? 💳
```

**Se Morno:**

```
Atendente: Ótimo! Vamos dar um trial de 7 dias.

✅ Acesso total ao software
✅ Nenhum cartão necessário
✅ Suporte dedicado
✅ 7 dias para testar sem compromisso

Qual seu melhor email? 📧
```

**Se Frio:**

```
Atendente: Sem problemas! Aqui temos:

📖 Guia: "Como escolher software certo"
📺 4 vídeos de 5 min explicando diferenças
💬 Chat direto comigo qualquer dúvida

Qual tópico te interessa? 🤔
```

### Passo 7: Processamento de Pagamento

**Se Cartão/PIX:**

```
Cliente clica link pagamento (Stripe)
     ↓
Insere dados (cartão ou PIX)
     ↓
Pagamento processado (instantâneo)
     ↓
Email com confirmação + acesso
```

**Se Boleto:**

```
Cliente clica "Gerar Boleto"
     ↓
Boleto gerado + enviado por email
     ↓
Cliente paga pela internet banking
     ↓
TupãSoft detecta pagamento (até 24h)
     ↓
Acesso ativado + email confirmação
```

### Passo 8: Ativação e Onboarding

```
Dentro de 2 horas após pagamento:

✉️ Email Recebido:
├─ Bem-vindo!
├─ Chave de licença / Acesso
├─ Guia de implementação (PDF)
├─ Vídeo tutorial (link YouTube)
├─ Telefone suporte (nosso)
└─ Agendamento sessão setup (opcional)

☎️ Se solicitado:
├─ Chamada 30 min com especialista
├─ Setup inicial do software
├─ Treinamento para 1º uso
└─ Resolução dúvidas
```

### Passo 9: Suporte Contínuo

```
Semana 1: Onboarding
├─ Configuração inicial
├─ Importação de dados
└─ Treinamento

Semana 2-4: Ramp-up
├─ Suporte às dúvidas
├─ Ajustes
└─ Feedback

Mês 2+: Operacional
├─ Suporte via WhatsApp
├─ Atualizações
└─ Upsell (recursos adicionais)
```

### Passo 10: Retenção e Upsell

```
Se MENSAL:
├─ Renovação automática todo mês
├─ Cliente pode cancelar anytime
└─ Foco: Reter máximo

Se LICENÇA:
├─ Sem renovação necessária
├─ Suporte por 1 ano (ou indefinido)
└─ Foco: Vender complementar

Upsell Oportunidades:
├─ "Ficou interessado em [outro software]?"
├─ Desconto para cliente atual (10-15%)
└─ Integração entre softwares
```

## 🎯 Estratégia de Preços

### Precificação

```
MODELO A: SEM MARKUP (Apenas Comissão)
├─ Preço para cliente = Preço do Fabricante
├─ TupãSoft ganha comissão
├─ Vantagem: Melhor preço, confiança
└─ Seu ganho: 20-30% do preço

MODELO B: COM MARKUP (Futuro)
├─ Preço = Preço Fabricante + 5-10%
├─ TupãSoft ganha comissão + markup
├─ Risco: Cliente compra direto depois
└─ Ganho: +5-10% margem
```

**Recomendação Ano 1:** Modelo A (confiança)
