## 📋 Regras da Comunidade

### ✅ COMPORTAMENTO ESPERADO (Obrigatório)

```
1. RESPEITO MÚTUO
   ├─ Trate todos com dignidade
   ├─ Ouça ativamente quem discorda
   ├─ Reconheça e aprenda com erros
   └─ Celebre sucessos alheios

2. COMUNICAÇÃO PROFISSIONAL
   ├─ Linguagem clara e respeitosa
   ├─ Sem insultos, xingamentos ou menosprezo
   ├─ Crítico == feedback, não ataque
   └─ Respeite privacidade e confidencialidade

3. INCLUSÃO ATIVA
   ├─ Bem-vindo a todos (raça, gênero, etc)
   ├─ Não exclua ninguém por aparência/fala/sotaque
   ├─ Adapte linguagem para iniciantes
   └─ Mencione pronouns se confortável

4. FOCO NA MISSÃO
   ├─ Contribua com qualidade
   ├─ Respeite o tempo dos outros
   ├─ Mantenha conversas relevantes
   └─ Assuma responsabilidade
```

### ❌ COMPORTAMENTO NÃO ACEITÁVEL (Proibido)

#### Violência & Ameaças

```
Proibido:
├─ Ameaças físicas ou morte
├─ Incitação à violência
├─ Doxxing (revelar dados pessoais)
└─ Stalking ou intimidação

Consequência: Ban permanente + denúncia à polícia (se crime)
```

#### Assédio

```
Proibido:
├─ Assédio sexual ou avanços não convidados
├─ Comportamento predatório
├─ Chantagem ou exploração
└─ Linguagem sexualmente explícita

Consequência: Ban 1º vez + apelação impossível 2º offensa
```

#### Discriminação & Ódio

```
Proibido:
├─ Linguagem odiosa (étnica, racial, religiosa)
├─ Discriminação por gênero, sexualidade, deficiência
├─ Caricaturas ou estereótipos
└─ Exclusão baseada em identidade

Consequência: Ban imediato
```

#### Abuso & Toxicidade

```
Proibido:
├─ Xingamentos, insultos pessoais
├─ Ataques ad hominem ("você é burro")
├─ Sarcasmo destrutivo ou ridicularização
├─ Mobilização de grupo contra alguém
└─ Gaslighting ou manipulação

Consequência: Warn 1x → Mute 1 semana → Ban
```

#### Spam & Abuso de Plataforma

```
Proibido:
├─ Spam de links, mensagens ou conteúdo
├─ Self-promotion excessivo
├─ Publicidade não autorizada
├─ Tentativa de phishing, fraude ou hack
└─ Compartilhamento de malware

Consequência: Ban imediato + denúncia ISP
```

#### Propriedade Intelectual

```
Proibido:
├─ Compartilhar código sem permissão
├─ Violar licenças de software
├─ Roubar ideias de outros
├─ Impersonação de identidade
└─ Falsificar informações

Consequência: Ação legal + ban
```

## 🚀 Processo de Denúncia

### Como Reportar

#### 1️⃣ **Menor (Aviso amigo)**

Comportamento leve (desfoco, tom rude)?
→ Mensagem privada amigável: _"Ei, acho que você não quis, mas isto me machucou..."_

#### 2️⃣ **Médio (Report informal)**

Comportamento inapropriado (spam, xingamento)?
→ Reporte para moderadores via formulário ou DM

#### 3️⃣ **Grave (Report formal)**

Assédio, ameaças, discriminação?
→ Email diretamente: **conduct@tupansoft.com.br**

### O Quê Incluir

```
Reporte deve ter:
├─ Data, hora e timezone
├─ Link/screenshot da mensagem
├─ Pessoas envolvidas
├─ Contexto (thread)
├─ Por que violou código?
└─ Pedido: warn / mute / ban / outro
```

### Formulário de Denúncia

```
De: seu.email@example.com
Para: conduct@tupansoft.com.br
Assunto: [CODE OF CONDUCT] Denúncia de violação

Corpo:
─────────────────────────
Tipo de violação: [ ] Assédio [ ] Spam [ ] Ódio [ ] Outro
Data: __ / __ / 2026
Hora: __ : __ (Timezone: ___)
Pessoa(s) envolvida(s): @usuario1, @usuario2
Link/Screenshot: [LINK ou DESCRIÇÃO]
Explicação: [Descreva o que aconteceu]
─────────────────────────
```

## ⚖️ Processo de Investigação

### Timeline

```
1. RELATO (T+0)
   └─ Denúncia recebida via email/formulário

2. ACUSAR (T+24h)
   └─ Confirmamos recebimento
   └─ Informamos à pessoa acusada

3. INVESTIGAR (T+2-7 dias)
   ├─ Entrevistamos testemunhas
   ├─ Coletamos logs/evidências
   └─ Revisamos histórico

4. DECIDIR (T+7-14 dias)
   ├─ Time delibera
   ├─ Informamos à acusadora
   └─ Informamos ao acusado

5. APELAR (T+14-21 dias)
   ├─ Parte pode apelar a decisão
   ├─ Revisão por arbitrador neutro
   └─ Decisão final
```

### Princípios de Investigação

```
✅ Equidade
   └─ Ambas as partes são ouvidas

✅ Transparência
   └─ Processo documentado

✅ Privacidade
   └─ Informações sigilosas protegidas

✅ Presunção de Inocência
   └─ Culpa deve ser provada

✅ Proteção à Vítima
   └─ Vítima não sofre retaliação
```

## 🎯 Consequências (Progressivas)

### Nível 1: AVISO

**Quando:** Primeira violação leve (spam, tom rude)  
**Ação:**

- Mensagem privada explicando problema
- Oportunidade de resposta

**Exemplo:** "Vejo que você postou 20 mensagens iguais. Isso é spam. Evite repetir."

### Nível 2: MUTE (1 Semana)

**Quando:** Segunda violação ou primeira grave (xingamento)  
**Ação:**

- Silenciado por 7 dias
- Pode ler, mas não pode postar
- Aviso claro do por quê

**Exemplo:** Usuário xingou moderador → 1 semana mute

### Nível 3: SUSPENSÃO (1 Mês)

**Quando:** Terceira violação ou segunda grave  
**Ação:**

- Conta suspensa por 30 dias
- Conteúdo removido sob análise
- Email explicativo

**Exemplo:** Usuário voltou a spam após mute → 1 mês suspenso

### Nível 4: BAN PERMANENTE

**Quando:** Quarta violação ou violação séria (assédio, ódio, ameaça)  
**Ação:**

- Conta deletada
- IP banido da plataforma
- Possível denúncia legal
- Publicado na "banned list" (nome, não detalhes)

**Exemplo:** Usuário fez propostas sexuais não consensuais → BAN

### Especial: BAN IMEDIATO

Violações que resultam em ban SEM progressão:

```
├─ Ameaça de morte
├─ Assédio sexual
├─ Linguagem discriminatória (étnica/racial)
├─ Hack/tentativa de invasão
├─ Doxxing
├─ Tráfico/exploração infantil
└─ Crime em geral
```

## 🛡️ Proteção de Vítimas

### Durante Processo

```
✅ Anonimato
   └─ Nome da vítima não revelado ao acusado (exceto crime)

✅ Sem Retaliação
   └─ Proibido responder com hostilidade

✅ Suporte
   └─ Recursos de saúde mental (contatos)

✅ Privacidade
   └─ Conversa não publicada publicamente
```

### Após Resolução

```
✅ Se Culpa Provada
   └─ Vítima não encontra acusado em espaços compartilhados
   └─ Conta acusado muted/moved

✅ Se Inocência Provada
   └─ Vítima não sofre punição
   └─ Pode apelar se sentir injustificada
```

## 📢 Responsabilidade de Moderadores

### Deveres

```
✅ Moderadores devem:
├─ Ser imparciais e justos
├─ Responder em até 24h
├─ Manter confidencialidade
├─ Documentar tudo
├─ Recusar-se se conflito de interesse
└─ Denunciar crime à polícia (lei)
```

### Conflito de Interesse

Moderador não pode julgar se:

- Vítima é amigo/família
- Acusado é amigo/família
- Moderador foi envolvido na disputa
- Moderador tem relação comercial

→ Nesse caso: Delega para outro moderador

### Denúncia de Moderador

Moderador violou código?
→ Reporte para **CEO (ceo@tupansoft.com.br)** ou **conselho independente**

## 📖 Exemplos de Violações

### CASO 1: Spam Leve

```
Mensagem: "@todos comprem em http://spamsite.xyz"

Ação: NÍVEL 1 (Aviso)
└─ Mensagem removida
└─ Mensagem privada: "Por favor, sem self-promotion"
└─ If continua? → Nível 2
```

### CASO 2: Linguagem Ofensiva

```
Mensagem: "Que [slur racial] comentário burro"

Ação: NÍVEL 2 (Mute 1 semana)
└─ Mensagem deletada
└─ Email: "Linguagem discriminatória proibida"
└─ If voltõe? → Nível 3
```

### CASO 3: Assédio Sexual

```
Mensagem: "Hey babe, quer ir pra meu Airbnb? 😏"

Ação: NÍVEL 4 (BAN IMEDIATO)
└─ Conta deletada
└─ Vítima informada
└─ Possível denúncia DPCA (cyberstalking)
```

### CASO 4: Ameaça de Morte

```
Mensagem: "Vou encontrar você e matar"

Ação: BAN IMEDIATO + POLÍCIA
└─ Conta deletada + IP banido
└─ Denúncia Polícia Civil (ameaça)
└─ Registrado em "banned forever" list
```
