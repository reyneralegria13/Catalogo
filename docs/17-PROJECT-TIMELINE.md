## 📋 Dependências e Riscos

### Dependências Críticas

```
1. Documentação 100% (Lei 15 Abr)
   └─ Sem isso: Não pode lançar

2. Deploy servidor produção (Lei 30 Abr)
   └─ Sem isso: Não tem infraestrutura

3. Buy-in de 3+ fornecedores (Lei 31 Mai)
   └─ Sem isso: Catálogo vazio no lançamento

4. Soft launch validação (Lei 14 Mai)
   └─ Sem isso: Bugs críticos no lançamento
```

### Riscos Potenciais

```
RISCO 1: Atraso na aprovação (Legal/Compliance)
├─ Impacto: 2-4 semanas de delay
├─ Probabilidade: 20%
└─ Mitigação: Revisar cedo + leverage experts

RISCO 2: Fornecedores não confirmam a tempo
├─ Impacto: Lançamento vazio (fracasso)
├─ Probabilidade: 30%
└─ Mitigação: Parceiros de backup ja contactados

RISCO 3: Traffic < esperado (conversão baixa)
├─ Impacto: Receita 50% menor
├─ Probabilidade: 40%
└─ Mitigação: Pivot quick para content + partnerships

RISCO 4: Bug crítico logo após lançamento
├─ Impacto: Reputação damage, churn
├─ Probabilidade: 10%
└─ Mitigação: Testes thorough + rollback plan
```
