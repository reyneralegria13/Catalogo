### Definições

**Crítica:** Sistema inoperante, perda de dados  
**Alta:** Feature principal não funciona  
**Média:** Problema menor, workaround existe  
**Baixa:** Sugestão, UI, documentação

## 🔧 Troubleshooting Rápido

### "Erro 504 — Gateway Timeout"

**Causas:**

- Servidor sobrecarregado
- Conexão lenta
- Firewall bloqueando

**Soluções:**

1. Aguarde 5 minutos
2. Tente novamente
3. Verifique internet (speedtest.net)
4. Se problema persistir → Contate suporte

---

### "Senha incorreta"

**Causas:**

- Caps Lock ativado
- Espaços em branco
- Usuário errado
- Senha expirada

**Soluções:**

1. Verifique Caps Lock
2. Use "Recuperar senha"
3. Contacte support@tupansoft.com.br

---

### "Arquivo não encontrado"

**Causas:**

- Arquivo foi movido/deletado
- Caminho de pasta errado
- Permissões insuficientes

**Soluções:**

1. Verifique caminho do arquivo
2. Procure em Arquivos recentes
3. Restaure do backup
4. Se problema continua → suporte técnico
